Для сайта интернет-магазина был написан код CSS для задания стиля (footer.css, form.css, widget.css). 
Для того, чтобы активировать стили необходимо в панеле администратора WordPress во вкладке "Дополнительные стили" добавить данные коды.

Для того, чтобы добавить поиск в header (function.php), необходимо в панеле администратора WordPress во вкладке "Внешний вид", затем "Редактировать темы", в файл function.php добавить данный код.
